# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AQI_taly
- Real-time air quality monitoring for Italy
- Interactive map with PM10 and PM2.5 data
- AI-powered pollution forecasting using Prophet
- Contextual layers (fires, industrial facilities)
- BBOX spatial filtering for optimized performance
- Smart icon states for pollution source visualization
- Pollution source analysis with AI
- Debounced map updates for smooth performance
- Lazy loading for efficient data fetching
- Dual view modes (real-time and forecast)
- Comprehensive station details and historical data
- Professional floating UI components

### Backend Features
- Flask REST API with caching
- Integration with ARPA, OpenWeatherMap, incendioggi.it
- E-PRTR industrial facilities database
- Prophet-based time series forecasting
- Spatial filtering with BBOX support
- Pollution source analysis engine

### Frontend Features
- React 18 with Hooks
- React-Leaflet interactive maps
- Custom floating UI components
- Optimized rendering with memoization
- Debounced API calls
- Smart icon states with animations
- Responsive design
